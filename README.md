# Solitary Pursuits


### Hexo 

#### Setup

```
npm install
```

#### Serve locally

```
hexo server
```

#### Create new post

```
hexo new hello-world
```

#### Create new page

```
hexo new page hello-world
```

#### Create new draft

```
hexo draft hello-world
```


#### Deploy

```
hexo clean && hexo build && hexo deploy
```
